## Sticky Ads 

Плагин для вывода прилипающей рекламы внутри контента для мобильных устройств.

## Инструкция

Как пользоваться плагином, а также, зачем он нужен - подробно рассказано в инструкции:

http://seocherry.ru/ads/kak-ispolzovat-plagin-sticky-ads-shortkody-ruchnaja-vstavka-cherez-ads-inserter-flat-realbig/

Плагин добавляет в админке новый раздел Sticky Ads (в меню слева), в котором можно добавлять свои блоки рекламы. 
В поле редактора необходимо вставить рекламный блок из интерфейса РСЯ или Адсенс (или другой рекламной сети). Есть возможность настроить высоту контейнера - поле для ввода высоты в пикселях расположено под редактором кода рекламного блока.

Там же присутствует уникальный шорткод, с помощью которого можно добавить данный блок в любое место (скопироват-вставить) в тексте записи, или в другой плагин, например, Ads Inserter. 

После публикации рекламного блока, он будет отображаться при редактировании записей. На панели инструментов визуального редактора добавляется кнопка, которая включает/выключает панель вставки шорткода в текст. В панели выводятся все опубликованные блоки рекламы в виде заголовков, по которым можно понять, что есть что. 

## Дополнительная информация

В плагине реализована проверка обновлений непосредственно из админки. Сообщение о появлении новых версий будет отображаться в списке всех установленных плагинов. 

Планов по дальнейшему развитию и расширению функционала нет и не будет - плагин выполняет одну простую задачу и большего от него не требуется.
